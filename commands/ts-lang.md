# /ts-lang [langCode]

사용자가 이 명령어를 입력하면 다음 단계를 따라 설정을 진행하세요:

1. 만약 `langCode` 인자가 없다면 현재 설정된 언어(`~/.claude/claude-trans.json`의 `language`)를 알려주세요.
2. 예시 언어 목록을 보여줍니다:
   - `ko` (Korean)
   - `ja` (Japanese)
   - `zh` (Chinese)
   - `en` (English - 번역 비활성화와 동일)

3. 업데이트 시 `~/.claude/claude-trans.json`의 `language` 필드를 해당 코드로 변경하세요.
4. "번역 언어가 [langCode](으)로 설정되었습니다."라고 응답하세요.
